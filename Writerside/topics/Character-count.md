# Character count

To show a character or word count use either `->withCharacterCount()` or `->withWordCount()`.

## Characters

```PHP
Canvas::make('Content')
    ->withCharacterCount(),
```

## Words

```PHP
Canvas::make('Content')
    ->withWordCount(),
```
